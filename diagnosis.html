Quick Fixes to Try

1. Reset Browser Zoom:

  • Windows/Linux: Ctrl + 0
  • Mac: Cmd + 0
  • Or check browser zoom in address bar

2. Check Display Scaling:

  • Windows: Settings → Display → Scale
  • Mac: System Preferences → Displays → Resolution
  • Should be at 100% or default

3. Try Different Browser:

  • Test in Chrome, Firefox, Safari
  • Try incognito/private mode
