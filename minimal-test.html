Font Size Test

This page tests if fonts are displaying at normal sizes.

Pixel Sizes:

14px - Small text (should be quite small)

16px - Normal text (should be readable but not large)

18px - Slightly larger (should be noticeably bigger)

20px - Large text (should be clearly large)

REM Sizes:

0.875rem - Small text

1rem - Normal text

1.25rem - Large text

Expected Results:

  • 14px text should be small but readable
  • 16px text should be normal reading size
  • 20px text should be noticeably larger
  • If ALL text appears huge, there's a browser zoom issue
